matplotlib is a python library used to build pretty charts from jmh output http://matplotlib.org/

Installing matplotlib
-Make sure you have python 2.7 installed

Install matplotlib on Windows
-Install Microsoft Visual C++ Compiler for Python 2.7 
--Download from https://www.microsoft.com/en-us/download/details.aspx?id=44266
--Run the installer
-Install Matplotlib
--download Win32 wheel from https://pypi.python.org/pypi/matplotlib/
--Run "pip install <FILENAME>"

Install matplotlib on Linux
-sudo apt-get install python-matplotlib
